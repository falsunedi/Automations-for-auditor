# Automations for Auditor

This repository contains simple automation tools designed for UX auditors and researchers. Each tool is organized in its own folder with dedicated documentation to guide you through installation and usage.

## Available Automations

### [Anatomicrop](./Anatomicrop)
A macOS Quick Action that automatically crops 120 pixels from the bottom of images.


## Installation

### Download the Repository

**Option 1: Using Git**
```bash
git clone https://github.com/falsunedi/Automations-for-auditor.git
cd Automations-for-auditor
```

**Option 2: Download ZIP**
1. Click the green "Code" button at the top of this page
2. Select "Download ZIP"
3. Extract the downloaded file to your preferred location

### Individual Tool Setup

Each automation tool has its own installation requirements. Navigate to the tool's folder and follow the instructions in its README.md file.
